# User Guide

This guide is for people who want to use Pondview to explore data, ask questions, and build dashboards.

If you are looking for developer and architecture docs, use the [Technical Guide](/guide/).

## Start here

1. Read [What Pondview Is](/user/what-is-pondview) for the quick product overview.
2. Follow [Getting Started](/user/getting-started) to configure an AI key and bring in data.
3. Use [Main Workflows](/user/workflows) to learn the everyday flow from chat to dashboards.
4. Check [FAQ](/user/faq) and [Troubleshooting](/user/troubleshooting) when you get stuck.
